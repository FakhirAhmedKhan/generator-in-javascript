# 🎲 Random Name Generator in JavaScript

A simple and fun random name generator built using plain JavaScript. This project generates random names from a predefined list, perfect for games, apps, or just for fun!

## 🚀 Features

- Generates a new name with a single click  
- Customizable name list  
- Lightweight and fast  
- Easy to understand code – great for beginners!

## 🛠️ Technologies Used

- HTML  
- JavaScript


## 🔧 How to Use

1. Download or clone this repository  
2. Open `index.html` in your browser  
3. Click the "Generate" button to get a new random name!
